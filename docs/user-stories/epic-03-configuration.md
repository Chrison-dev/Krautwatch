# Epic 03 — Configuration

## US-006 · Configure Download Settings

> As a user, I want to configure where downloads are saved and how many run in parallel so that the app fits my setup.

**Acceptance Criteria:**
- Configurable: download directory, max concurrent downloads
- Settings are persisted across restarts
- Invalid paths are caught with a clear validation message
- Defaults work out of the box — no required setup to get started
